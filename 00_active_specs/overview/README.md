## 📖 目录简介

本目录包含 Clotho 项目的高层次介绍和入门指引文档。这些文档不涉及具体技术实现细节，而是关注项目的核心理念、设计哲学和整体架构概念。

## 📚 文档列表

### 1. 愿景与哲学

- **文件**: [`vision-and-philosophy.md`](vision-and-philosophy.md)
- **简介**: Clotho 项目的根本指导思想和设计哲学，包括"凯撒原则"、"混合代理"等核心概念。
- **阅读建议**: 新用户必读，理解项目为何存在以及要解决什么问题。
- **关联文档**: 主索引 [`../README.md`](../README.md)，快速介绍 [`../quick-intrFoduction.md`](../quick-introduction.md)

### 2. 架构原则

- **文件**: [`architecture-principles.md`](architecture-principles.md)
- **简介**: 系统架构的核心原则和设计决策，包括三层分离、确定性编排、状态管理等。
- **阅读建议**: 在阅读具体实现前，了解系统的设计约束和基本原则。
- **关联文档**: 核心架构目录 [`../core/`](../core/)，运行时目录 [`../runtime/`](../runtime/)

## 🔗 导航指南

### 从哪里开始？

如果您是**新用户**或**项目管理者**：

1. 首先阅读根目录的 [`../quick-introduction.md`](../quick-introduction.md) 快速了解项目
2. 然后阅读本目录的 [`vision-and-philosophy.md`](vision-and-philosophy.md) 理解设计理念
3. 最后浏览主索引 [`../README.md`](../README.md) 了解完整文档结构

如果您是**技术评估者**：

1. 阅读 [`vision-and-philosophy.md`](vision-and-philosophy.md) 了解项目定位
2. 直接跳转到感兴趣的专题领域（如核心架构、协议规范等）
